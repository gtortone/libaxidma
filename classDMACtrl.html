Detailed Description

AXI DMA controller.

Manage AXI DMA controller providing methods for status control (halt, run, reset) and data transfer from/to FPGA

◆ rx()

bool DMACtrl::rx ( uint32_t  timeout = 0)

Start a DMA S2MM data transfer.

DMA mode (direct or scatter-gather) is checked and related method is invoked. In case of long wait time during scatter-gather transfer (when timeout is not specified) the transfer is switched from buffer (all descriptors) to single block descriptor.

Parameters
timeouttimeout value (us) for non-blocking call (0: infinite)
Returns
true: data transfer completed
false: timeout expired
